# Intro to AI Projects  

By Luke Matsunaga  

This repository contains projects completed as part of an Intro to Artificial Intelligence course, focusing on search algorithms and machine learning concepts.

## Projects

### 1. 8-Puzzle Solver
- Implemented search algorithms including:
  - Uniform Cost Search (UCS)
  - A* with Misplaced Tile heuristic
  - A* with Manhattan Distance heuristic  
- Explores state-space search and optimal pathfinding  

Location: [8 Puzzle Solver](8_Puzzle_Solver/README.md)

### 2. Data Set Feature Selection
- Implemented feature selection algorithms:
  - Forward Selection  
  - Backward Elimination  
- Evaluates feature subsets using nearest neighbor classification  
- Demonstrates search in feature space and model evaluation  

Location: [Dataset Feature Selection](Dataset-Feature-Selection/README.md)

## Key Concepts
- State-space search  
- Heuristic design  
- Machine learning fundamentals  
- Feature selection and evaluation  
- Algorithm performance analysis  

## Technologies Used
- C++ (C++17)  
- Standard Template Library (STL)  

## Purpose
These projects demonstrate foundational AI concepts including problem formulation, search strategies, and data-driven model optimization.
